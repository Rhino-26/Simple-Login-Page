🔐 Simple Login Page (UI)

A modern and fully responsive login page UI built using HTML, CSS, and JavaScript. This project focuses on clean design, user experience, and essential authentication interface features commonly used in real-world applications.

The login screen includes email and password fields with proper validation, ensuring users enter correct and secure input. A show/hide password toggle improves usability, while error messages provide instant feedback for invalid entries.

The interface is designed with a sleek and minimal layout, making it visually appealing and easy to use across all devices. Additional options like “Remember Me” and “Forgot Password” enhance functionality and simulate a real authentication flow.

🚀 Features:
- 📧 Email and password input fields
- 👁️ Show/Hide password toggle
- ⚠️ Form validation with error messages
- ☑️ “Remember Me” checkbox
- 🔗 “Forgot Password” link
- 🎨 Clean, modern UI design
- 📱 Fully responsive layout (mobile, tablet, desktop)
- 💡 Smooth interactions and transitions
- 🎯 UI/UX Highlights
- Minimal and professional design
- Centered login card layout
- Clear input feedback and error states
- Smooth hover and focus effects
- Mobile-first responsive design
  
💻 Technologies Used:
- HTML – Structure
- CSS – Styling & responsiveness
- JavaScript – Validation & interactivity
  
📱 Responsive Design

The login page is optimized for all screen sizes:

- Mobile: Compact, touch-friendly layout
- Tablet: Balanced spacing and readable inputs
- Desktop: Centered card with enhanced spacing
